# 😊 AI Sentiment Analyzer

A simple yet powerful AI-powered web application that analyzes the sentiment of text using Natural Language Processing (NLP).

---

## 🚀 About the Project

This project helps to automatically detect whether a given text is:

- 👍 Positive
- 👎 Negative
- 😐 Neutral (sometimes based on input)

It is built using **Python** and **Streamlit**, making it beginner-friendly and interactive.

---

## ✨ Features

✔ Real-time sentiment analysis  
✔ Simple and clean UI  
✔ Fast processing using TextBlob  
✔ Beginner-friendly project  
✔ Easy to run and modify  

---

## 🛠️ Technologies Used

- Python 🐍  
- Streamlit 🌐  
- TextBlob 🤖  
- Pandas 📊  
- Matplotlib 📈
